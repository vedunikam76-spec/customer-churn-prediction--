# 📉 Customer Churn Prediction

## 📌 Problem Statement
Predict customer churn for a subscription-based service using historical customer data.

## 🛠️ Approach
- Features used: customer demographics, usage behavior, subscription details.
- Algorithms tested:
  - Logistic Regression
  - Random Forests
  - Gradient Boosting
- Evaluated using accuracy, precision, recall, and ROC-AUC.

## 📊 Sample Output
### Example Input:
Customer: Age = 32, Monthly Usage = Low, Tenure = 6 months, Complaints = Yes

### Predicted Result:
**Churn Likely**

### Model Performance:
- Gradient Boosting Accuracy: **89%**
- Random Forest Accuracy: **86%**
- Logistic Regression Accuracy: **82%**
